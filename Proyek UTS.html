Preface: Middle Examination Project - Generalized Linear Model

My project, a hallmark of my journey in actuarial science, revolves around exploring the mechanics of a Generalized Linear Model (GLM)—particularly a Poisson regression—on a dataset from Sweden’s motor insurance claims in 1977. In a world where data holds the power to predict and protect, I’ll dive into the intricacies of motor insurance, risk classification, and claims, revealing patterns that drive decisions in the industry.

In this analysis, the number of claims will serve as the primary response variable. Using the glm() function in R Studio, I’ll utilize the faraway library and fit a Poisson model with a log-link function. My goal is to evaluate the statistical significance of various covariates—kilometers driven, geographical zone, car make, no-claims bonus, and others—and quantify their impact on the number of claims.

Nevertheless, for the purpose of this examination, I’ll proceed with Poisson modeling, keeping in mind that my mentor’s focus is on testing my ability to understand and build this model.

Now, for the cool part—let’s break down the statistical methods I’ll use:

  1. Model Selection: To determine whether using all covariates or just a subset is better, we’ll compare models using metrics like AIC (Akaike Information Criterion). The model with the lower AIC is considered better, meaning it balances fit and complexity. Likelihood-ratio tests can also be used to compare nested models and see if removing certain covariates significantly worsens the fit.

  2. Wald Test: The Wald test is used to assess the significance of individual regression coefficients. It’s similar to the t-test but specifically tailored for GLMs. In R, the summary(glm) output will show z-values (from Wald tests), which indicate whether each coefficient differs significantly from zero. Large z-values (or small p-values) suggest that the covariate is statistically significant.

  3. Goodness-of-Fit (GoF): For Poisson models, deviance and Pearson chi-square are typical GoF measures. They assess how well the model fits the data. If overdispersion is present, these statistics will be higher than expected under a Poisson assumption, which signals the model might not fit well.

  4. Generalized Fitted Statistics (GFS): This refers to using measures like the deviance residuals or Pearson residuals to assess the model. Residual analysis will help check for patterns in the errors, indicating potential model misfits.

  5. Residual Analysis: Plotting residuals vs. fitted values helps check model assumptions. If the residuals exhibit a pattern, it suggests the model is not capturing all the variance in the data, possibly due to overdispersion or omitted variables.

  6. Visualization and Confidence Intervals: I will create visualizations of the regression results, including:

    • Confidence intervals for the regression coefficients (betas), which indicate the range within which we can be confident the true parameter lies.

    • Prediction intervals for the expected number of claims based on the fitted model, giving a visual understanding of uncertainty in predictions.
